# Shotgun Code

[English version](README.md)

Десктопное приложение для сборки контекста кода для AI-ассистентов. Собирайте, фильтруйте и экспортируйте кодовую базу в форматах, оптимизированных для LLM.

## Возможности

### Управление проектами

- Открытие локальных папок или клонирование с GitHub/GitLab
- Список недавних проектов с быстрым доступом
- Отслеживание изменений файловой системы в реальном времени

### Файловый менеджер

- Древовидное представление с раскрытием/сворачиванием
- Множественный выбор через Ctrl/Shift
- Быстрые фильтры: по расширению, git-статусу, размеру
- Правила игнорирования: поддержка .gitignore + свои паттерны
- Предпросмотр файлов с подсветкой синтаксиса

### Сборка контекста

- Выбор файлов для включения в контекст
- Подсчёт токенов с отображением лимитов модели
- Форматы вывода: Markdown, XML, JSON, Plain text
- Генерация манифеста с деревом файлов
- Опция удаления комментариев

### Экспорт

- Копирование в буфер обмена
- Экспорт в PDF (для AI с загрузкой файлов)
- Разбиение больших контекстов на части
- История контекстов с восстановлением

### Интеграция с AI

- Провайдеры: OpenAI, Gemini, OpenRouter, LocalAI, Qwen
- Выбор модели для каждого провайдера
- Кэширование ответов для идентичных запросов
- Потоковые ответы

### Git-интеграция

- Переключение веток
- Индикаторы статуса файлов (изменён, добавлен, удалён)
- Фильтрация по git-статусу
- Клонирование репозиториев GitHub/GitLab

## Установка

Скачайте последний релиз для вашей платформы:

- Windows: `shotgun-code-windows-amd64.exe`
- macOS: `shotgun-code.app.zip`
- Linux: `shotgun-code-linux-amd64`

Или соберите из исходников (требуется Go 1.24+, Node.js 20+):

```bash
# Установка Wails CLI
go install github.com/wailsapp/wails/v2/cmd/wails@latest

# Клонирование и сборка
git clone https://github.com/WhiteBite/shotgun_code.git
cd shotgun_code
wails build
```

## Использование

1. Откройте папку проекта или клонируйте репозиторий
2. Выберите файлы в проводнике (используйте фильтры)
3. Проверьте количество токенов и скорректируйте выбор
4. Скопируйте контекст в буфер или экспортируйте в PDF
5. Вставьте в ваш AI-ассистент

## Технологии

- Backend: Go, Wails
- Frontend: Vue 3, TypeScript, Pinia, Tailwind CSS
- Сборка: Vite, GitHub Actions
