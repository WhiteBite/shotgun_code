# Retrieval Policy

Порядок источников:
1) LSP/LSIF/SCIP символы и типы.
2) Code Graph: imports/call graph/references; соседи 1–2 хопа.
3) Тесты рядом (coverage/refs).
4) Эмбеддинги (Qdrant/Faiss) — доп. сигнал, отфильтрованный графом.

Упаковка:
- Manifest → файлы по алфавиту; «умные вырезки» для больших файлов.
- Детеминизм: стабильный порядок/заголовки; контроль токен-бюджетов.
