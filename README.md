# Shotgun Code

Инструмент для разработки с поддержкой AI и управления задачами.

## Разработка

### Запуск в режиме разработки

```bash
./dev.ps1
```

Для просмотра полного вывода (включая отладочную информацию):

```bash
./dev.ps1 -Verbose
```

### Требования

- Go 1.21+
- Node.js 18+
- Wails CLI v2.10.1+

### Известные особенности

**Избыточный вывод Wails CLI**: При запуске в режиме разработки Wails v2.10.1 выводит отладочную информацию генератора TypeScript биндингов, включая:
- Список всех структур домена (`KnownStructs: ...`)
- Сообщения `Not found: time.Time`

Это нормальное поведение генератора биндингов и не влияет на работу приложения. Вывод автоматически фильтруется в скрипте `dev.ps1`. Для просмотра полного вывода используйте параметр `-Verbose`.

**Рекомендация**: Обновите Wails до последней версии для потенциального устранения избыточного вывода:

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

Подробная информация в [DEVELOPMENT.md](DEVELOPMENT.md).

## Структура проекта

- `backend/` - Go backend с Wails
- `frontend/` - Vue 3 frontend
- `config/` - Конфигурационные файлы
